# Home

-- This project is related to the bike accessories pit shop.

## Products

-- In this product page, there are some products like helmets, gloves, and bike lights. Each product has a name, price, and description.
-- The products are displayed in a grid format, and users can click on each product to view more details.

## Cart

-- The cart page allows users to view the items they have added to their cart. Users can update the quantity of each item or remove items from the cart. The total price is also displayed on this page.
